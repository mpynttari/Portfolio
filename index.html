ABOUT ME

Olen neljännen vuoden Software Engineering-opiskelija LUT-yliopistossa. Tietotekniikkaa päädyin opiskelemaan, koska olen aina ollut kiinnostunut juuri tietokoneiden software-puolesta, välittämättä paljoa niiden fyysisestä sisällöstä.

Opintojen aikana kiinnostukseni on erityisesti herännyt web-developmentia kohti sekä muuhun front-end puolen koodaukseen. Visuaalisuus ja asioiden ulkonäkö ovat lähellä sydäntäni, ja koenkin olevani jokseenkin taiteellinen sielu. Tämä on tärkein syy miksi olen innostunut enemmän projektien front-end -puolesta back-endin sijaan.

Opinnoissa ja niiden ohella kokemusta on karttunut aihealueista:

  • React.js
  • HTML+CSS
  • Javascript
  • Git / versionhallinta
  • Node.js
  • Express.js
  • Java
  • Android Studio
  • Linuxin perusteet

Yllä mainittujen lisäksi myös videopeli-devaamisesta ja pikseligrafiikoiden piirtämisestä olen saanut pintaraapaisun muutamasta Code Camp- ja Hackathon-tapahtumasta.

Olen innokas oppimaan enkä kaihda uusien teknologien tai koodauskielten/koodikirjastojen harjoittelemista.
